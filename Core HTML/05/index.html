Front-end Framework
React: A JavaScript library for building user interfaces and can be used for developing single-page applications.
Back-end Framework
Express.js: A Node.js framework for building scalable and fast server-side applications.
Database
MongoDB: A NoSQL database for storing and retrieving data in a flexible and scalable manner.
Programming Language
JavaScript: A high-level language used for both front-end and back-end development, known for its versatility and dynamic nature.
Version Control
Git: A distributed version control system for tracking changes and collaborating on code repositories.
API Gateway
NGINX: A web server and reverse proxy that can be used as an API gateway to manage incoming requests and route them to the appropriate backend services.
Authentication
OAuth 2.0: An authorization framework for securing APIs and providing secure access to protected resources.
CSS Preprocessor
Sass: A CSS preprocessor that allows for writing more efficient and modular CSS code.
Package Manager
npm (Node Package Manager): A package manager for JavaScript that allows for easy installation and management of dependencies.
Testing Framework
Jest: A JavaScript testing framework for writing unit tests and integration tests for front-end and back-end code.
